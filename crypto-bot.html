BTC Auto-Trade Bot 운영 중

DeepSeek v4 Pro 기반 완전 자율형 비트코인 단기 트레이딩 시스템. 1시간 단위로 시장 데이터를 분석하고 GPT 수준의 추론으로 매매 결정을 내린다. 조코딩 GPT-Bitcoin 프로젝트 아키텍처 기반, DeepSeek 마이그레이션 완료.

실험용 시스템

  • 이 봇은 교육 및 실험 목적으로 제작되었다. 수익성을 입증하는 백테스트 데이터가 존재하지 않는다.
  • 잃어도 괜찮은 금액 외에는 절대 투입하지 말 것.
  • 실제 자금 투입 전 최소 3개월 드라이런(모의 거래) 필수.
  • 이 봇은 JSM 장기 투자 전략(investment-agent.mdc)을 보완하는 소액 실험용 위성 시스템이다.

시스템 아키텍처

5개 모듈이 파이프라인으로 연결되어 1시간마다 자동 실행된다.

메모리

1시간 주기 실행 파이프라인

시장 데이터 수집 (Module 1)
업비트 API에서 30일 일봉 + 24시간 시간봉 OHLCV 수집. SMA, EMA, RSI, MACD, Stochastic, Bollinger Bands 등 7개 기술적 지표 계산. 호가창(Orderbook) 데이터 포함.
뉴스 및 감성 데이터 수집 (Module 2)
SerpApi로 "Bitcoin" 관련 최신 뉴스 10건 크롤링. Alternative.me에서 공포탐욕지수(Fear & Greed Index) 30일치 시계열 수집.
DeepSeek v4 Pro 추론 (Module 4)
수집된 모든 데이터 + 과거 거래 기록 + instructions.md 프롬프트를 DeepSeek에 전송. DeepSeek은 6단계 의사결정 워크플로우를 거쳐 JSON 형식으로 buy/sell/hold 결정 반환. 응답은 3단계 fallback 파서로 검증. 실패 시 지수 백오프 5회 재시도.
거래 실행 (Module 5)
JSON 결정 검증 (비율 상한 30%, 잔고 체크, 환각 방지). 공포탐욕지수 연동 비율 조정 (극단적 공포 시 매수↑, 극단적 탐욕 시 매도↑). 업비트 API로 시장가 주문 실행. 수수료 0.05% 감안.
기록 및 재귀적 개선 (Module 6)
모든 결정을 SQLite DB에 영구 저장. 다음 1시간 루프에서 과거 결정을 컨텍스트로 재주입 → 자기 교정 학습. 3연속 손실 시 서킷 브레이커 자동 발동.

기술적 지표 상세

DeepSeek이 매매 판단에 사용하는 7가지 지표

SMA / EMA
10일 이동평균
EMA > SMA 상향돌파 → 매수
EMA < SMA 하향돌파 → 매도
RSI
14일 상대강도지수
30↓ 과매도 → 매수 기회
70↑ 과매수 → 매도 검토
MACD
12/26/9 이동평균 수렴발산
MACD ↑ Signal선 돌파 → 매수
MACD ↓ Signal선 돌파 → 매도
Stochastic
%K=14, %D=3
20↓ 과매도 → 반등 기대
80↑ 과매수 → 조정 예상
Bollinger Bands
20일, 2σ 표준편차
하단밴드 터치 → 저평가 매수
상단밴드 터치 → 과열 매도
Fear & Greed Index
30일 시계열 (Alternative.me)
0-25 극단적 공포 → 역발상 매수
75-100 극단적 탐욕 → 익절 검토

실행 방법

1. 드라이런 (모의 거래)

업비트 API 키 없이도 DeepSeek 판단까지 전부 테스트 가능. 실제 주문은 실행되지 않는다.

# .env 설정 DRY_RUN=true # UPBIT 키는 비워둬도 됨 # 실행 python autotrade.py
2. 실전 거래

업비트에 원화 입금 후 API 키 발급. DRY_RUN=false로 전환.

# .env 설정 UPBIT_ACCESS_KEY=... UPBIT_SECRET_KEY=... DRY_RUN=false # 실행 (백그라운드) nohup python3 -u autotrade.py > output.log 2>&1 & # 로그 확인 tail -f output.log

AWS EC2 배포 (향후)

인스턴스 t2.micro (프리 티어, 첫 12개월 무료)
OS Ubuntu 22.04 LTS
타임존 sudo ln -sf /usr/share/zoneinfo/Asia/Seoul /etc/localtime
보안 SSH(22)만 인바운드 허용. 업비트 API IP 화이트리스트에 Elastic IP 등록.
