# JSM Wiki v3 + Coach 사용법

---

## Coach란?

네 인생 전반을 이끌어주는 개인 코치. 연구/학습/운동/투자/일상 결정까지 모든 걸 커버한다.

---

## 5가지 말만 기억하면 됨

"읽어서 위키에 반영해줘"
→ raw/나 2_Sources/ 파일을 위키 페이지로 변환

"저장해줘"
→ 대화 내용을 위키 페이지로 생성

"결정 기록해줘"
→ 중요한 결정 단계별로 기록 (synapse)

"점검해줘"
→ 위키 건강 상태 검사 (lint)

그 외 질문은 그냥 말하면 됨
→ 위키 검색 먼저, 없으면 웹 검색
→ 위키에 있더라도 불확실하면 웹으로 팩트 체크
→ 빠르게 변하는 주제면 항상 웹 검색 병행
→ 답변이 길면 "저장할까?" 제안

## 메모 시스템

빠른 메모: "적어놔", "기록해놔", "메모" 같은 키워드와 함께 말하면 JSM-memo.md에 저장
코어 컨텍스트: .cursor/rules/core-context.mdc -- 항상 자동 로드되는 핵심 정보
영구 지식: 3_Wiki/ 위키 페이지 (v3 프론트매터)

---

## 절대 하면 안 되는 것

X raw/, 2_Sources/ 파일 수정 금지 (소스는 불변)
X 위키 페이지는 3_Wiki/ 폴더에만 생성, 하위 폴더 금지
X domains 프론트매터 직접 작성 금지 (자동 계산)
X 페이지 연결: 프론트매터 related 대신 본문에 "builds on [[페이지]]" 형식으로

---

## 폴더 구조

raw/          새 소스 (Git 추적)

3_Logs/       동결된 레거시 로그
